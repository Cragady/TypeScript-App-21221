Somewhat of a Reddit clone

## Main Technologies

* React
* TypeScript
* GraphQL
* URQL/Apollo
* Node.js
* PostgreSQL
* MikroORM/TypeORM
* Redis
* Next.js
* TypeGraphQL

# Knitting a tsconfig file:

* `npx tsconfig.json`

# Some Simple Workflows:

## File Watchers and Server:

In separate terminals: 

* `yarn watch`
* `yarn dev`

## Migrations:

* `npx mikro-orm migration:create`
* `npx mikro-orm migration:create --run`
